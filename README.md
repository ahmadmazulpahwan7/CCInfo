# CCInfo
CCInfo (Credit Card Information). Get information from credit card number.
# Installation
```bash
$ pkg install git
$ pkg install python2
$ git clone https://github.com/ITermSec/CCInfo.git
$ cd CCInfo
$ python2 install.py
```
